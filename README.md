# 📊 Retail Inventory & Outlet Insights Dashboard (Power BI)

## 📌 Overview

This project presents an interactive Power BI dashboard analyzing retail inventory data to uncover patterns in item distribution, pricing trends, and outlet characteristics.

## 🎯 Objectives

* Analyze item distribution across different outlet types
* Understand impact of outlet size and location tier
* Identify category-wise pricing trends
* Highlight top 10 expensive items
* Enable dynamic filtering using slicers

## 📊 Key Insights

* Supermarket Type1 has the highest item distribution
* Medium-sized outlets contribute the most inventory
* Tier 3 locations show higher product availability
* Categories like Starchy Foods and Household items have higher average pricing
* A small group of items fall under the premium price range

## 🛠 Tools Used

* Power BI
* Data Cleaning & Transformation
* Data Visualization
